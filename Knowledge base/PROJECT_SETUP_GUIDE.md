## Step-by-Step Setup

### Step 1: Create a New Project
1. Go to claude.ai
2. Click **"Projects"** in the sidebar
3. Click **"Create Project"**
4. Name it (e.g., "Marketing Coach - [New Business Name]")

### Step 2: Add Custom Instructions
1. In your new project, click the **gear icon** or **"Set custom instructions"**
2. Copy the ENTIRE contents of `Custom_Instructions_Marketing_Coach.md`
3. Paste it into the custom instructions box
4. **IMPORTANT:** Before saving, customize it:
   - Replace "Lucy" with the new person's name throughout
   - Update team size, goals, and current situation
   - Adjust target audiences if different
   - Modify compliance terms if the business has different restrictions

### Step 3: Upload Knowledge Base
1. Click **"Add knowledge"** in the project
2. Upload `Knowledge_Base_Financial_Services.md`
3. Upload the Blue Screen presentation PDF
4. **IMPORTANT:** Customize the knowledge base:
   - Update the profile section with new person's details
   - Adjust goals and metrics
   - Modify scripts with appropriate names and details

### Step 4: Set User Preferences (Optional)
In Settings > Profile, you can add preferences like:
- "I am a visual learner, when explaining concepts, I learn well with analogies."
- Any other communication preferences

## What to Customize for a New Person

### In Custom Instructions:
- [ ] Name (replace "Lucy" everywhere)
- [ ] Business stage and experience level
- [ ] Team size
- [ ] Social media platforms they use
- [ ] Income/sales goals
- [ ] Primary constraints (day job, schedule, etc.)
- [ ] Target audiences (may differ by market)
- [ ] Any business-specific compliance rules

### In Knowledge Base:
- [ ] Profile section
- [ ] Goals table
- [ ] Script templates (names, locations, personal details)
- [ ] Content examples
- [ ] Tools & resources they actually use

## Tips for Best Results

1. **Start each session with context** — Tell the coach what you're working on today
2. **Be specific about prospects** — Share details so the coach can personalize messages
3. **Report wins and challenges** — This helps the coach calibrate advice
4. **Use tiered activities** — Match your daily effort to your available time
5. **Follow the compliance rules** — The restricted terminology list is there for a reason
